# Investec Swagger Skills

This folder contains agent-friendly “skills” for each Investec OpenAPI/Swagger document in this repository.

## Goals
- Provide short, predictable API workflows
- Keep Swagger files canonical in ../swagger/
- Enable AI agents to discover APIs easily
